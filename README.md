# 🚀 User Directory Dashboard

A modern and interactive User Directory Dashboard built using React. This application fetches user data from an external API and provides real-time search, sorting, and detailed user views with a clean and user-friendly interface.

---

## 📌 Features
- Display users in a structured table (Name, Email, Phone, Company)
- Search users by Name or Email (client-side filtering)
- Sort users by Name and Company (ascending/descending)
- Click on a user to view complete profile details
- Clean and responsive UI

---

## 🛠️ Tech Stack
React (Vite) • React Router DOM • Custom CSS • REST API (JSONPlaceholder)
